# Codex Telegram Bridge v0.1.0

First public version of the project.

## Highlights

- Local Telegram bridge for Codex on Windows
- Per-agent routing, queueing, and thread reuse
- Safe split between public example config and private local config
- Windows scheduled-task support
- Included Codex skill for setup and operation

## Included

- Bridge runtime in TypeScript
- Windows helper scripts
- Example config files
- MIT license
- `telegram-codex-bridge` skill

## Known limitations

- Requires local Codex installation and authentication
- Uses CLI execution instead of `app-server`
- Stores runtime state in JSON
